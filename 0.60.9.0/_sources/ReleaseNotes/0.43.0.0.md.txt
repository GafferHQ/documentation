0.43.0.0
========

Features
--------

- GafferVDB : Added PointsGridToPoints node and visualisation
  of VDB point grids ([#2386](https://github.com/GafferHQ/gaffer/issues/2386)).

Improvements
------------

- Appleseed : Added support for Capsules generated by the Encapsulate
  node ([#2402](https://github.com/GafferHQ/gaffer/issues/2402)).
- ImageReader ([#2380](https://github.com/GafferHQ/gaffer/issues/2380)) :
  - Reimplemented to use Gaffer's native cache instead of
    OpenImageIO's cache, giving speedups of 20-30% in some cases. This
    also paves the way for introducing deep images to Gaffer.
  - Added initial support for multi-part OpenEXR files.

Fixes
-----

- ValuePlug : Fixed problems when serialising compound plugs ([#1500](https://github.com/GafferHQ/gaffer/issues/1500), [#2395](https://github.com/GafferHQ/gaffer/issues/2395)).
  - BoxPlugs were being serialised with `setValue()` calls for the leaf plugs
    _and_ the top level plug itself.
  - FormatPlugs were incorrectly attempting to serialise a `getValue()` call
    at the top level even when input connections existed at the leaf level.
    This prevented a common pattern for the dispatch of image networks from
    working.
- Camera node : Fixed "Copy From Viewer" tool menu item ([#2396](https://github.com/GafferHQ/gaffer/issues/2396)).
- SceneInspector : Fixed transform component display ([#2396](https://github.com/GafferHQ/gaffer/issues/2396)).
- Expression : Fixed bug caused by non-serialisable input plugs ([#2395](https://github.com/GafferHQ/gaffer/issues/2395)).
- CollectImages/Merge/Offset : Fix out of bounds input tile accesses
  ([#2380](https://github.com/GafferHQ/gaffer/issues/2380)).
- TabbedContainer : Fixed an incompatibility with Qt5 ([#2426](https://github.com/GafferHQ/gaffer/issues/2426)).
- PythonCommand : Exposed imath module to the execution command ([#2428](https://github.com/GafferHQ/gaffer/issues/2428)).
- CompoundPlug : Added compatibility config for legacy gfr files ([#2429](https://github.com/GafferHQ/gaffer/issues/2429)).

API
---

- Serialisation : Added support for nested classes in `classPath()` method ([#2400](https://github.com/GafferHQ/gaffer/issues/2400)).
- Plug : Remove deprecated ReadOnly flag ([#2401](https://github.com/GafferHQ/gaffer/issues/2401)).
- Moved PathMatcher/PathMatcherData/PathMatcherDataPlug from GafferScene to Gaffer ([#2404](https://github.com/GafferHQ/gaffer/issues/2404)).
- Filter : Removed Result enum. Use IECore::PathMatcher::Result instead ([#2404](https://github.com/GafferHQ/gaffer/issues/2404)).

Breaking Changes
----------------

- GafferScene :
  - Removed AlembicSource node. Use a SceneReader instead ([#2397](https://github.com/GafferHQ/gaffer/issues/2397)).
  - Removed AlembicPath and AlembicPathPreview.
- GafferCortex : Removed TimeCodeParameter support. This parameter type
  has been removed from Cortex ([#2427](https://github.com/GafferHQ/gaffer/issues/2427)).
- GafferTest : Removed SphereNode ([#2399](https://github.com/GafferHQ/gaffer/issues/2399)).
- ValuePlugSerialiser : Removed `valueNeedsSerialisation()` virtual
  method ([#2395](https://github.com/GafferHQ/gaffer/issues/2395)).
- OpenImageIOReader : Removed cache related methods ([#2380](https://github.com/GafferHQ/gaffer/issues/2380)).
- Removed Plug::ReadOnly flag. Use MetadataAlgo instead ([#2401](https://github.com/GafferHQ/gaffer/issues/2401)).
- ValuePlugSerialiser : Removed flagsMask argument from `repr()` method ([#2401](https://github.com/GafferHQ/gaffer/issues/2401)).
- Filter : Removed Result enum. Use IECore::PathMatcher::Result instead ([#2404](https://github.com/GafferHQ/gaffer/issues/2404)).
